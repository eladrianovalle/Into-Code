Ruby Methods: Date.strftime

Wednesday, March 25, 2015

In my Ruby Skillcrush course, we've been using the Date object in many of our projects. The Date object is incredibly powerful, with a mutitude of methods that give us a wealth of information to use with our apps. But the output we get is often in a format that isn't suitable for us to display to human users. Enter the strftime method. 

Date.strftime("%args") => "date string"

Using strftime, we can take the output of our Date object and transform it into a nicely formatted string. And more so, we have a number of formatting directives we can pass through to the output to create differently formatted outputs.

Here are a few examples for a Date object d = Date.now representing today, March 25, 2015:

Date in "Month Day, Year" format 

d.strftime("%B %d, %Y") => "March 25, 2015"

Date in "Weekday, Month Day, Year" format 

d.strftime("%A, %B %d, %Y") => "Wednesday, March 25, 2015"

Date in "MMDDYYYY" format 

d.strftime("%m%d%Y") => "03252015"

Note that you can include other characters with the directives in your argument, like spaces and commas, even entire words, that will be included in your output.

For example:

d.strftime("Created on %m/%d/%Y")   #=> "Created on 03/25/2015"

Mike Buckbee built a cool little tool called "For a Good Strftime" that lets you pick and choose the format you want for your output and it'll tell you what directives you'll need to pass through to the strftime argument. Strftimer is a sweet tool that allows you to input a date string in the format you wish to employ, and it'll output the exact declaration you need to acheive it. For an extensive list of all the possible directives you can pass through to strftime, check the Ruby docs here, this resource at APIdock, or this post by Akshay Mohite.

The Date, Time, and DateTime objects all behave similarly, both as Ruby objects and in regards to the strftime method, but they each have their own specific uses and slight differences, so make sure you use the right one for your specific application.
